INTRODUCTION |
-------------
Andhrimmir: The Digial Recipe Book

Andhrimnir is a repository of recipes that users can browse through,
search for specific recipes, add to the repository, and leave
comments for other users to read.

Users either log in as a basic user, moderators, or admins. Users are
able to view recipes, leave comments, and request new recipes be
added. Moderators are charged with curating the comments to remove
hateful and/or inappropriate messages. Admins have the responsibility
of approving new recipes and removing troublesome users that cause
frequent issues.

USER MANUAL |
------------

**********************************************************************
CREATE USER ACCOUNT *
Press ‘ Create User Account ‘ button, which switches to the sign in page, enter first name, username and password in the field, Press ‘create’. If success message is shown, the account is created.
*********************

**********************************************************************
LOGIN *
Go to the ‘user log in’ page, enter the username and password, Press ‘login’ button below. 
*******

**********************************************************************
SEARCH/FILTER *
In the Recipe Repository page, on the right,  type in the name of the recipe wanted or ingredient in ‘search here’ field. 
To narrow the search by cuisine type, choose wanted cuisine type in grey box called ‘Filter by cuisine type’. press filter button.
To narrow the search by  Calorie, enter minimum and maximum calorie. Press filter button.
To reset search, press ‘reset Search and Filter Results to default’ button above search bar.
